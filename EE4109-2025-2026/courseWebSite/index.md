# EE4109 2025-2026: Structured Electronics Design

**Instructors:** Chris Verhoeven and Anton Montagne
**Institution:** TU Delft
**Last Updated:** 16/12/2025

## Course Overview

This MSc course on "Structured Electronics Design" uses a practical educational approach centered on designing a hearing loop receiver, progressing through design aspects step-by-step.

## Educational Structure

The course alternates between two formats:
- **Odd-numbered lectures:** Introduce design aspects with accompanying homework assignments
- **Even-numbered lectures:** Review the completed work

Students conclude with an oral group examination on negative feedback amplifier design in CMOS technology.

## Prerequisites

Students should have completed:
- EE3C11 or the homologation program
- Proficiency with simulation tools: KiCAD, LTspice, NGspice, SLiCAP
- Python programming
