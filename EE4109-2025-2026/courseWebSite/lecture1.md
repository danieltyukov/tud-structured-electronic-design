# Lecture 1: Introduction EE4109 (11-11-2025)

## Course Overview
This TU Delft course teaches structured electronics design using a systems engineering approach. The capstone project involves designing a hearing aid telecoil amplifier, with assessment through an oral group exam focused on design decisions and verification.

## Key Learning Objectives
Students must demonstrate "complete and correct motivation of the design of the signal path" and verify designs using SLiCAP and SPICE. Biasing considerations are mandatory; bias circuit design is optional but rewarded.

## Design Project: Hearing Loop Receiver
The course centers on a practical hearing aid amplifier application. Students work in groups (3-7 members) throughout the semester.

## Primary Resources

**Presentations:**
- Course Introduction and Hearing Loop System introduction
- Background materials from Sonion (hearing aid manufacturer)

**Technical Documentation:**
- Receive coil, microphone, battery, and loudspeaker datasheets
- DSP specifications
- Historical patent references

**Software & Exercises:**
- SLiCAP installation required (design tool)
- RC filter example project with noise/settling-time specifications
- Linear time-invariant system analysis examples
- ABCD matrix parameter test bench

## Course Structure
Lectures run from mid-November 2025 through January 2026, covering amplification principles, MOS modeling, feedback stages, and frequency compensation across 14 sessions.
