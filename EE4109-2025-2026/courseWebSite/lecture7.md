# Lecture 7: Feedback Stages (02-12-2025)

## Overview
This lecture covers feedback stages as negative feedback amplifiers with single-stage controllers, specifically using biased CS stages or balanced variants.

## Key Topics

### 1. Introduction to Feedback Stages
- **Resource**: "Introduction to feedback stages" presentation available at analog-electronics.tudelft.nl
- **Video**: EE4109 2020 7_1 on YouTube demonstrating foundational concepts

### 2. Common Drain (CD) Stage
The CD stage represents a nonenergic feedback configuration with a CS stage controller. Key resources include:
- Presentation: "Common Gate Stage" PDF
- Two instructional videos (7_2 and 7_3)
- SLiCAP Python files for stability and frequency compensation analysis (CDstage.zip)

### 3. Common Gate (CG) Stage
Similarly structured as feedback architecture, the CG stage emphasizes "error-reduction capabilities" and behavioral modifications from negative feedback application.
- Related presentation and video (7_4) available

### 4. Additional Feedback Stages
Advanced configurations combining active feedback, passive feedback, indirect feedback, and balancing techniques covered in presentation 7_5.

## Resources Available

**Presentations & Videos**: Multiple PDF presentations and YouTube videos (EE4109 2020 series)

**Downloads**:
- CD stage analysis files (CDstage.zip)
- CD stages notebook

**Posters**: CD stage and amplifier stages reference materials

## Group Exercise Questions
Students address implications of using CD/CG stages as input/output stages and bulk effect influences on voltage transfer.
