System Architecture

Stage A1: Coil-to-ADC Integrator

Design Targets

The receive coil transduces the induced magnetic field into a voltage signal. This signal must interface with the ADC input, maintaining voltage-to-voltage conversion characteristics.

Type: voltage to voltage

Grounded input, grounded output

$Z_{in} = \infty$ (Ideally)

$Z_{out} = 0$ (Ideally)

$V_{out,pp} = 636$ mV

Required Transfer Function

Since the coil output is proportional to the rate of change of magnetic flux, the design requires an integrating transfer characteristic. Using microphone specifications, the required transfer is:

$\frac{V_{out}}{V_{in}} = \frac{62.4\cdot 10^3}{s}$

Implementation Approach

Several topologies can achieve the required integration:

The single-stage three-port configuration offers advantages including superior noise performance, reduced current consumption, and smaller die area compared to four-port alternatives. The input resistance can be designed to provide the required termination.

Alternative approach: active feedback topology

Stage A2: Buffer Amplifier

type: Voltage to voltage

$V_{in,pp} = 636$ mV

$V_{out,pp} = 900$ mV

This stage amplifies the integrated signal to the ADC full-scale input for optimal resolution utilization. The modest 1.41V/V gain allows omission of this stage if the source impedance permits, trading dynamic range for area savings.

The input impedance (5.5k$\Omega$) creates a low-pass filter with the ADC input capacitance (10pF), yielding a cutoff frequency of 2.9MHz. This exceeds the signal bandwidth, ensuring the driving capability is sufficient.

Stage A3: DAC-to-Speaker Driver

Design Targets

Voltage to voltage

Single ended to differential

The DAC output impedance (10k$\Omega$) requires an input impedance significantly higher to minimize loading effects.

Common mode output voltage around 0V

$V_{in,pp} = 900$ mV

The required output amplitude is determined by the acoustic performance target. For an RMS voltage of 0.18V driving the loudspeaker to 109dBSPL (conservative relative to the 110dBSPL specification), the speaker voltage is:

$V_{LSP} = 10^{\frac{110-109}{20}}0.18=0.202$ V

Including the audio signal crest factor, the peak-to-peak speaker voltage becomes: $V_{LSP,pp} = 1.21$ V, which remains below both the supply voltage and typical component ratings.

Implementation Options

Option 1: Achieves higher input impedance at the cost of requiring a rail-to-rail output controller

Option 2: Implements high input impedance with a standard controller that does not require rail-to-rail operation
